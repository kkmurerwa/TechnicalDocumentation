Overview

Kotlin is a statistically typed language that was unveiled in 2011 by JetBrains as part of the Kotlin Project. Its syntax is not compatible with JAVA but the language relies on the JAVA Virtual Machine(JVM) and can also run JAVA code. Kotlin is compatible with up to JVM 11 as of the time of publishing this documentation(December 2018). Kotlin uses type inference to determine the type of values and expressions for which type has been left unstated. It incorporates the speed of C/C++ and the stability of JAVA to make an all-powerfull language that can be used for any day-to-day programming problems.
The lead developer of the Kotlin project, Dimitri Jemerov said they developed the language because no language had the features that they were looking for with the exception of Scala, which however, has a ver slow run-time. The name of the language comes from an island, Kotlin, near St. Petersburg. This was done in imitation of JAVAs name derived from the Indonesian island of Java. The first stable version of the language was released in February 2016, . JetBrains has committed to backwards compatibility from this version. The latest version of Kotlin is v1.3 that was released in October 2018.
Kotlin is now supported by Google as of October 2017 alongside C/C++ and JAVA as the main languages used in Android programming.

Getting Started

In order to get started with Kotlin, you are advised to visit the Official Kotlin Documantation site. Once done, you require a Kotlin IDE. This can be an online IDE such as the one on Tutorials Point or a locally installed one. The lastest build of the online compiler v1.3 can be downloaded at Kotlin.
A shorter method would be to install the IntelliJ IDEA which comes precompiled with the latest Kotlin compiler. Other IDEs that can be used include Visual Studio Code and Eclipse. You however have to install the needed plugins in order to write Kotlin code on the latter IDEs.

Basic Data Types

Kotlin language has several data types. These include literals and numbers.

Numbers

There are numerous number types in Kotlin. These types and their properties are as listed below.
  1. Double - 64 bits
  2. Float - 32 bits
  3. Long - 64 bits
  4. Int - 32 bits
  5. Short - 16 bits
  6. Byte - 8 bits

Literals

Kotlin supports a wide number of literals. These include:
  1. Hexadecimals - 0x0F
  2. Binaries - 0b011010111
  3. Characters
  4. String
  5. Boolean
  6. Arrays
Kotlin, however, does not support Octal numbers.

Variable Declaration

Kotlin variables are declared using the keyword val Below are some variable declarations and instantiations. Data types in Kotlin, like Java, have to be declared explicitly as shown in the code below.
val text: Str = "I am text"
val integer: Int = 10
val float: Float = 10.254
val hex_Number: Hex = 0xFF
